Analysis

Functions for analyzing properties of sensing matrices.

CompSense.mutual_coherence — Function
mutual_coherence(A)

Compute the mutual coherence of matrix A:

μ(A) = max_{i≠j} |⟨aᵢ, aⱼ⟩| / (‖aᵢ‖₂ ‖aⱼ‖₂)

The mutual coherence is the maximum absolute normalized inner product between distinct columns. It bounds the spark and is key for recovery guarantees: if ‖x‖₀ < (1 + 1/μ(A)) / 2, then x is the unique sparsest solution.

Arguments

  • A::AbstractMatrix: Sensing matrix (m × n)

Returns

  • Float64: Mutual coherence in [0, 1]

Example

A = randn(50, 200)
mu = mutual_coherence(A)

References

  • Donoho & Elad (2003), "Optimally sparse representation"

CompSense.babel_function — Function
babel_function(A, k)

Compute the cumulative coherence (Babel function) μ₁(k) for matrix A.

μ₁(k) = max_i  max_{|Λ|=k, i∉Λ}  Σ_{j∈Λ} |⟨aᵢ, aⱼ⟩| / (‖aᵢ‖₂ ‖aⱼ‖₂)

The Babel function provides tighter recovery guarantees than mutual coherence. If μ₁(k-1) + μ₁(k) < 1, greedy algorithms like OMP exactly recover any k-sparse signal.

Arguments

  • A::AbstractMatrix: Sensing matrix (m × n)
  • k::Int: Sparsity level

Returns

  • Float64: Babel function value μ₁(k)

Example

A = randn(50, 200)
mu1 = babel_function(A, 10)

References

  • Tropp (2004), "Greed is Good: Algorithmic Results for Sparse Approximation"
source
CompSense.spark — Function
spark(A)

Compute the spark of matrix A: the smallest number of linearly dependent columns.

spark(A) = min { ‖x‖₀ : Ax = 0, x ≠ 0 }

The spark provides an exact (but NP-hard in general) sparsity threshold: if ‖x‖₀ < spark(A)/2, then x is the unique sparsest solution to Ax = b.

Arguments

  • A::AbstractMatrix: Sensing matrix (m × n), must be small (n ≤ 20 recommended)

Returns

  • Int: Spark of A

Example

A = randn(5, 8)
s = spark(A)

Notes

  • Computing the spark is NP-hard in general. This function enumerates all subsets and is only practical for small matrices (n ≤ ~20).
  • Throws ArgumentError if n > 30 to prevent accidental combinatorial explosion.

CompSense.column_coherence_matrix — Function
column_coherence_matrix(A)

Compute the full Gram matrix of normalized columns of A.

G[i,j] = ⟨aᵢ, aⱼ⟩ / (‖aᵢ‖₂ ‖aⱼ‖₂)

Diagonal entries are 1. Off-diagonal entries measure pairwise column correlation. Useful for visualizing coherence structure (e.g., as a heatmap).

Arguments

  • A::AbstractMatrix: Sensing matrix (m × n)

Returns

  • Matrix{Float64}: n × n coherence Gram matrix

Example

A = randn(50, 200)
G = column_coherence_matrix(A)
# heatmap(abs.(G)) for visualization
